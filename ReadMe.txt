How to get the data ?

I have attached the text file for reference.

If you want to do infromation retrieval with similar kind of data.

1. Go to the pubmed website.
2. Login with account details.
3. In search bar type your dataset of interest(mine was university of new mexico papers)
4. Click on format--->Pubmed file
5. Click on save file as text
6. A text file will be downloaded
7. Attach the file while running the code in the prompt given by the google colab cell
8. Perform classification for the data and get top 5 - 10 similar papers.
